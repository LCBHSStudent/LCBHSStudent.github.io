void MainWindow::init ( const QString &  mltPath)

Initialises the main window.

参数
MltPath(optional) path to MLT environment
Url(optional) file to open
clipsToLoad(optional) a comma separated list of clips to import in project

If Url is present, it will be opened, otherwise, if openlastproject is set, latest project will be opened. If no file is open after trying this, a default new file will be created.
