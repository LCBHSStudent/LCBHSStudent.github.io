protectedpure virtual

Performs a resize of the given item. Returns true if the operation succeeded, and otherwise nothing is modified This method is protected because it shouldn't be called directly. Call the function in the timeline instead. If a snap point is within reach, the operation will be coerced to use it.

参数
sizeis the new size of the item
rightis true if we change the right side of the item, false otherwise
undoLambda function containing the current undo stack. Will be updated with current operation
redoLambda function containing the current redo queue. Will be updated with current operation
